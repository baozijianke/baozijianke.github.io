欢迎来到我们的中心

Florid艺术教育机构力求通过对艺术课程的学习，提高艺术修养，提高对美的感知和认识，
通过多样性的音乐表达模式，让学员在音乐中获得快乐，
通过对多样乐器的体验，帮助提高逆境商数(Adversity Quotient)，培养更好的表达能力及社交能力。

Some fun facts 优势

0 台专业设备

全部采用日本雅马哈钢琴，圆润的音色，丰富的表现范围和精准的控制性能，带来更加专业的音乐表现。

0 个舞蹈教室

选用安全性，减震性以及反弹性能超强的专业舞蹈地板，领先的减震保护系统，完美的保护舞蹈者运动安全。

0 个多媒体教室

配备133英寸1080P蓝光高清投影以及全球领先的哈曼卡顿数字影院系统。

0 位专业教师

本中心拥有一支国内一流的教师队伍，以四川音乐学院在职老师和一批具有留学经历的研究生，以及来自意大利、西班牙、波兰等外籍教师为班底，设立各学科不同教学模式板块，灵活多变的教学机制为每个学生量身设计学习内容，确保高质量的教学效果。

教学总监牟一娜

现任四川音乐学院声乐系钢琴艺术指导、钢琴教研室主任、四川省钢琴学会常务理事、四川省考级委员会评委。1990年就读于四川音乐学院附中，2000年以优异的成绩毕业于四川音乐学院钢琴系留校至今。 从大学开始经常参加各项演出和受邀担任国内国外比赛的钢琴伴奏，2008 年赴俄罗斯圣彼得堡音乐学院学习，历年所授学生多次获得奖项，具有丰富的演出和教学经验，并多次申请四川省级科研课题，在核心刊物上发表论文数篇。

教学副总监常力

2006年－2010年就读于中国传媒大学，获得音乐传播本科学位 2010-2012年就读于香港教育学院，师从梁保华教授，主要学习音乐教育，儿童教育，儿童心理学，钢琴演奏课程并获得音乐教育硕士学位； 2012年获得英皇（ABRSM）音乐理论考试五级； 2012年－2013年赴美国洛杉矶学习钢琴演奏，并受邀在Hills音乐培训机构担任钢琴老师。 2014年9月获得英国Trinity音乐学院颁发的钢琴演奏级一级证书 从大学开始经常参加各项演出和受邀担任国内国外比赛的钢琴伴奏，2008 年赴俄罗斯圣彼得堡音乐学院学习，历年所授学生多次获得奖项，具有丰富的演出和教学经验，并多次申请四川省级科研课题，在核心刊物上发表论文数篇。

音乐理论老师亚采克 JACEK GAPINSKI

亚采克，波兰著名法国号演奏家，音乐理论老师。 毕业于波兰比得哥熙音乐学院并获得音乐表演硕士学位。亚采克具有丰富和扎实的音乐知识和技巧。曾在多家著名的交响乐团担任法国号演奏者，例如托沦交响乐团。亚采克先后就职于国际教育学院，Goldenotes音乐机构，非常善于音乐理论的教授，特别是皇家考级和出国留学音乐理论。同时，也兼任法国号和钢琴老师。 从大学开始经常参加各项演出和受邀担任国内国外比赛的钢琴伴奏，2008 年赴俄罗斯圣彼得堡音乐学院学习，历年所授学生多次获得奖项，具有丰富的演出和教学经验，并多次申请四川省级科研课题，在核心刊物上发表论文数篇。
